Desktop Operating System

These are intended for use in a small office or home office with a limited number of users. These runs single-user applications and shares files and folders on on a small network.

Two ways to install multiple operating system on a single personal computer

  • Multi-boot
    • this refers to a custom boot loader configuration on a computer's primary drive which allows a user to select an operating system during startup. Disadvantage of this is that you need to restart the computer to switch operating system.
  • Virtualization
    • this enables a user to create an entire virtual machine inside an application. It is called virtualization software. It was installed in host operating system. and the operating system for each virtual machine is known as guest operating system. User can run two operating system simultaneously without rebooting the computer.
