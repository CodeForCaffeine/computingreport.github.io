Booting

This is a starting or restarting computer.

Two ways to boot a computer

  • Warm Boot
    • occurs when the computer is already ON and restart it without turning OFF the power.
  • Cold Boot
    • starting a computer that has been turned OFF

Features common to most operating systems

Three basic categories of operating system

  • Embedded Operating System
    • these are operating systems that are entirely stored within a device.
  • Stand-alone Operating System
    • also called desktop operating system control a single desktop or laptop computer.
  • Network Operating System (NOS)
    • are used to control or coordinate computers that are networked or linked together.

The operating system is often referred to as the software environment or software platform Almost all application programs are designed to run with a specific platform
